# 三寸光阴一个鑫

个人技术博客，基于 [VitePress](https://vitepress.dev/) 与 [@sugarat/theme](https://theme.sugarat.top/) 构建，记录学习笔记、面试题、开发随笔与个人作品。

**线上地址**：[https://hongyingxin.github.io/](https://hongyingxin.github.io/)

## 部署

项目通过 GitHub Actions 自动部署到 GitHub Pages。向 `master` 分支推送代码后，工作流会自动构建并发布站点。

工作流配置见 [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml)。
