# 实战场景

## 登陆凭据共享

通过全局数据通信，父应用通过`setGlobalData`修改全局数据，子应用通过`window.microApp.addGlobalDataListener`监听数据变化

```js
// 父应用
import microApp from '@micro-zoe/micro-app'

// setGlobalData只接受对象作为参数
microApp.setGlobalData({type: '全局数据'})

// 子应用
// 1. 获取初始值
window.microApp.setGlobalData({type: '全局数据'})

// 2. 绑定监听
window.microApp.addGlobalDataListener(handleGlobalData)
```

退出登录需要子应用通知父应用，在父应用统一清除。子应用通过`window.microApp.dispatch`提交，父应用通过自定义监听事件`@datachange`监听

```js
// 父应用
handleDataChange (e) {
  console.log('来自子应用的数据：', e.detail.data)
}

// 子应用
window.microApp.dispatch({ type: 'UNAUTHORIZED' })
```
