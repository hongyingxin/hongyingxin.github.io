# 组件设计

## head 组件

布局结构：分为左右两部分，左边是 logo 和导航菜单，右边是功能区域，包括搜索、主题、用户。

功能： 滚动时保持在顶部，背景模糊（毛玻璃效果），点击 logo 或导航菜单时，跳转到首页。点击导航栏，跳转到对应页面。

```text
┌─────────────────────────────────────────────────────────┐
│ Logo  Nuxt Movie   导航菜单              搜索 主题 用户 │
└─────────────────────────────────────────────────────────┘
```
