# 12月

## 12月30号

今天碰到一个需求，需要浏览器页面滚动截屏工具，我一直使用的`Snipaste`，但是这个工具只能截屏，不能滚动截屏，所以我就需要找其他工具。

Chrome 浏览器有一款非常不错的插件工具`GoFullPage`，目前是商店评分最高的截长图插件。一键点击，自动滚动截取，生成的图片符合我的需求。（ps：需要区别下李逵和李鬼，有款同名的`GoFullPage.net`，是假的，不要下载错了）

另外也学到一个骚操作，浏览`F12`截图。按下`Command + Option + i`打开开发者工具，按下`Command + Shift + p`打开命令面板，输入`Capture full size screenshot`(截取完整尺寸屏幕截图)，回车浏览器会自动下载一张完整的长图。
