# 26_6月

## 6月17日

在配置 Jenkins 的过程中，卡在`pnpm install`上，把依赖更新之后，还是不行。报错信息如下：

```bash
Cannot install with "frozen-lockfile" because pnpm-lock.yaml is not up to date with package.json
```

这才发现，没有把`pnpm-lock.yaml`文件也上传到仓库。

`pnpm-lock.yaml`文件是`pnpm`的锁定文件，记录了项目依赖的版本信息。为了保证生产环境的稳定，需要将`pnpm-lock.yaml`文件也上传到仓库，不能`.gitignore`忽略掉。
